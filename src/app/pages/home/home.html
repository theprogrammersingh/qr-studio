## How it works

  • QR generation runs in your browser via the qrcode library — no server round-trip.
  • Scanning uses the device camera and decodes locally; payloads never leave the page.
  • Recently generated and scanned codes are persisted in IndexedDB on this device only.
  • Install it from the address bar to use offline.
